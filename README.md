# Task Tracker

A simple task management application built with React.

Live Demo: (https://task-tracker-six-psi-47.vercel.app/)

## Overview

Basic to-do list application for task management. Originally designed as a Kanban board with drag-and-drop functionality and timer features - currently implements core task CRUD operations.

## Current Features

- Create new tasks
- Edit task details
- Delete tasks
- Clean, responsive UI with Tailwind CSS

## Technologies

- React.js
- Tailwind CSS
- JavaScript
- Vercel (deployment)

## Planned Enhancements

- **Kanban Board Layout:** Drag-and-drop functionality across "To Do," "In Progress," and "Done" columns
- **Functional Timers:** Working stopwatch for each task with start/pause/reset capabilities
- **Task Persistence:** Local storage or database integration to save tasks between sessions
- **Priority Levels:** Color-coded task priorities
- **Due Dates:** Date picker and deadline tracking

## Installation
```bash
git clone https://github.com/PavitrSri/task-tracker
cd task-tracker
npm install
npm start
```
